# Full-Body Strength for Beginners

## Goal

Describe a conservative strength template for users who need simple, repeatable
exercise selection.

## Exercise selection rules

- Favor compound patterns that train multiple muscle groups with simple setup.
- Pick stable exercises before highly technical or explosive variations.
- When possible, pair each lower-body movement with a push and a pull movement.
- Home plans should prioritize bodyweight, dumbbells, resistance bands, and
  chair or bench-supported variations.

## Set and rep guidance

- Most beginner strength work fits well in 2 to 4 sets of 6 to 12 repetitions.
- Use loads that allow clean technique and controlled tempo.
- Stop sets when technique clearly degrades instead of chasing extra repetitions.
- Rest about 60 to 120 seconds between moderate strength sets.

## Progression

- Add repetitions before load when the user has limited equipment.
- Add load only when the current rep range feels repeatable with solid form.
- If recovery is poor, reduce volume before removing training entirely.

## Substitution logic

- If a movement irritates a joint, swap to a simpler variation in a pain-free
  range of motion.
- If equipment is unavailable, preserve the movement pattern before chasing an
  exact exercise match.
